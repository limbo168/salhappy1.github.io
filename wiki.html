gavel Server Rules

By playing on the Brace Network, you automatically agree to follow all the rules listed below. Ignorance of the rules is not an excuse.

1. Chat Violations

No spamming, excessive cursing, bypassing the chat filter, advertising other servers, or directing toxicity towards other players.

2. Client Modifications (Hacking)

The use of X-ray, Aimbot, KillAura, Auto-Clickers, or any hacked client that gives an unfair advantage is strictly prohibited and results in a permanent ban.

3. Exploits & Glitches

Exploiting server bugs, duplication glitches, or block glitching is forbidden. Please report bugs on the forums for a bounty reward.

4. Griefing & Scamming

Griefing claimed land or scamming players in real-money transactions (IRL trading) will lead to a severe network ban. Scamming in-game items is allowed ONLY in Factions.

terminal Essential Commands

Command Description Aliases / Notes
/spawn Teleports you to the main server hub. /hub, /lobby
/tpa [player] Sends a teleport request to a player. Requires target to /tpaccept
/sethome [name] Sets a personal teleport location. Max homes based on Rank
/msg [player] [msg] Sends a private direct message. /whisper, /tell, /w
/ah Opens the Global Auction House. Safe trading menu
/jobs join Opens the Jobs menu to earn coins. Max 2 jobs at a time

payments Economy & Progression

Making Money

The main currency on the server is Coins. You can earn them through several methods:

  • Selling items to the server /shop
  • Player-to-player trading in the /ah
  • Completing Quests & Job actions
  • Voting daily for the server

In-Game Ranks

You don't need to spend real money to progress. Use your coins to buy in-game ranks via /rankup.

Peasant → Citizen $10,000
Citizen → Knight $50,000
Knight → Lord $250,000
